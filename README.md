# SIL
# SIL++: Deep Gated Semantic Injection for Collaborative Code Generation

Official PyTorch implementation of the paper **SIL++**. 
SIL++ acts as a portable fusion layer that bridges the misaligned latent spaces between discriminative encoders (CodeBERT) and generative models (CodeT5).

## 1. Environment Setup
```bash
conda create -n silpp python=3.10
conda activate silpp
pip install -r requirements.txt
